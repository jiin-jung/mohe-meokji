# Mohe Meokji

## Project Introduction
Mohe Meokji is an innovative project designed to enhance user experience in food discovery and restaurant management. 

## Development Period
The project is scheduled for development from January 2026 to June 2026.

## Environment
The application runs in a cloud-based environment utilizing various microservices for scalability and performance.

## Tech Stack
- **Frontend:** React.js, Redux
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Deployment:** AWS, Docker

## Architecture
The architecture follows a microservices approach with individual components handling specific functionalities, ensuring high availability and maintainability.

## Installation
1. Clone the repository: `git clone https://github.com/jiin-jung/mohe-meokji.git`
2. Navigate to the project directory: `cd mohe-meokji`
3. Install dependencies: `npm install`
4. Run the application: `npm start`

## Usage
After starting the application, navigate to `http://localhost:3000` in your web browser to access the interface.
